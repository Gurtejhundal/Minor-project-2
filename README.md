# Minor Project 2

Student Information System assignment completed by **Gurtejbir Singh**.

## How to Run

```bash
python main.py
```

## Features

- Stores student details using a list of dictionaries.
- Accepts name, roll number, age, and marks for 5 subjects.
- Validates marks between 0 and 100.
- Calculates total and average marks.
- Determines pass or fail status.
- Categorizes age as Child, Teenager, or Adult.
- Checks whether roll numbers are prime.
- Checks whether names are palindromes.
- Reverses roll numbers using a `while` loop.
- Shows highest and lowest marks.
- Prints Fibonacci numbers based on the number of students.
- Prints roll numbers from 100 to 200 that are multiples of both 3 and 5.
- Prints all prime numbers between 1 and 100.
